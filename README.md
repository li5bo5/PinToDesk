# PinToDesk - Windows桌面待办事项工具

一个极致轻量级的Windows 11桌面待办事项小工具，可以固定在桌面上，常驻系统后台，采用简洁的无边框半透明设计。

## 分区

- **标题栏**：显示窗口标题（"PTD"）
- **待办事项列表**：显示用户添加的待办事项，可进行编辑、删除、移动操作

## 功能区

- **标题栏**：鼠标按住标题栏空白处可以移动窗口
- **边角**：可以通过左下角和右下角自由调节窗口的大小
- **内容区**：列表显示待办事项
- **描述**：
  - 标题右侧是一个空白区域最右边只有最小化和关闭按钮，鼠标按住标题栏空白处可以移动窗口
  - 标题栏是四个文字："PTD"
  - 待办事项的添加、修改是直接在列表中的项目中进行
  - 待办事项只能输入一行10个字
  - 窗口显示5条待办，超过5条时，滚动显示

## 功能特点

1. **桌面固定**：
   - 窗口固定在桌面之上，始终可见
2. **窗口移动**：
   - 鼠标右键点击托盘按钮，选择调整，可以激活窗口调整
   - 激活后可用鼠标左键拖拽窗口到任意位置
   - 右键点击托盘按钮，选择退出调整，即可锁定位置
3. **待办事项管理**：
   - 双击待办列表空白区域即可新增
   - 悬停在待办事项右侧显示编辑、删除按钮
4. **界面设计**：
   - 半透明设计
   - 浅色主题
   - 简洁的列表显示，无复选框
5. **系统托盘**：
   - 系统托盘显示程序图标
   - 选项：
		调整/退出调整、显示/隐藏、开机启动/取消开机启动、退出
6. **数据持久化**：自动保存待办事项到本地文档文件夹
7. **开机启动**：可通过系统托盘菜单设置开机启动
8. **资源占用小**：轻量级设计，最小化系统资源消耗

## 使用方法

1. 运行程序后，窗口会显示在桌面右上角距离屏幕边缘一定位置
2. **添加待办事项**：双击待办列表空白区域，在待办事项列表最下方出现输入框，可输入待办事项内容，字数限制10中文字
3. **编辑待办事项**：双击某条事项即可编辑待办事项内容
4. **删除待办事项**：将鼠标悬停在待办事项上，右侧会出现删除按钮，点击即可删除
5. **待办事项排序**：鼠标点击待办事项按住不放，可拖动改变待办事项的显示顺序，松开鼠标即完成排序
6. **列表滚动**：当待办事项列表超出窗口高度时，会显示滚动条，可通过滚动条或鼠标滚轮滚动列表
7. **移动窗口**：鼠标右键点击托盘按钮，选择调整，可以激活窗口调整。激活后可用鼠标左键拖拽窗口到任意位置。右键点击托盘按钮，选择退出调整，即可锁定位置
8. **系统托盘操作**：
   - 双击托盘图标：
     - 显示/隐藏：显示/隐藏主窗口
   - 右键点击托盘图标：
     - 固定/取消固定
     - 开机启动/取消开机启动：切换程序是否随系统启动
     - 退出：完全退出程序
